{{ ui.help_label("닫힌 캔들 원칙", "closed_candle") }}

CMVA는 현재 형성 중인 캔들을 표시할 수 있지만, 저장, 특징, 모델, 쇼크 라벨, 레짐, 진단, 검증에는 닫힌 캔들만 사용합니다.

{{ ui.help_label("데이터 누적", "bootstrap_progress") }}

Binance Spot 과거 캔들은 심볼과 간격별로 누적되며, 누락 공백, 중복 타임스탬프, OHLC 일관성, 거래량 유효성, 미완성 캔들 유입 여부를 검사합니다.

{{ ui.help_label("변동성", "volatility") }}

실현 변동성, EWMA 변동성, 범위 기반 변동성, GARCH 예측 변동성은 선택한 캔들 간격의 수익률 변동성을 설명합니다.

{{ ui.help_label("GARCH 예측", "garch_forecast") }}

예측값은 sigma_{t+1|t}이며, 시점 t까지의 정보로 계산한 다음 캔들 변동성 예측입니다. 가격 방향 예측이 아닙니다.

{{ ui.help_label("추세", "trend_slope") }}

추세 특징은 로그 가격의 롤링 OLS와 진단용 t-통계량을 사용합니다. 관측 경로를 설명할 뿐 매수 또는 매도 지시가 아닙니다.

{{ ui.help_label("상관", "average_correlation") }}

상관, BTC 베타, PCA1 비중, 분산도는 시장이 함께 움직이는지 또는 심볼별로 분리되는지를 설명합니다.

{{ ui.help_label("쇼크 점수", "shock_score") }}

쇼크 점수는 t에서 관측된 수익률을 t 이전에 만든 변동성 예측과 비교합니다.

{{ ui.help_label("워크포워드 검증", "walk_forward") }}

CMVA에서 백테스트는 모델 검증을 뜻합니다. 예측 손실, 보정, 잔차 검정을 확인합니다.

{{ ui.help_label("룩어헤드 편향", "lookahead_bias") }}

t의 특징은 t까지의 데이터만 사용하고, 예측 평가는 forecast_{t|t-1}과 r_t를 정렬합니다.
